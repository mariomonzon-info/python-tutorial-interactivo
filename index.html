¿Qué es Python?

🌟 Python es como hablar con la computadora

Imagínate que quieres decirle a tu computadora qué hacer, pero en lugar de usar botones y clicks, le escribes instrucciones en un lenguaje que ambos entienden. ¡Ese es Python!

🔥 ¿Por qué Python es genial?

• Es fácil de leer - casi como escribir en español
• Es poderoso - puedes crear páginas web, juegos, aplicaciones
• Es popular - millones de programadores lo usan
• Es gratis - no cuesta nada empezar

# Tu primera línea de código Python
print("¡Hola, mundo!")

🎯 Características clave de Python

• Interpretado: No necesitas compilar, ejecutas directamente
• Tipado dinámico: Las variables no necesitan tipo explícito
• Indentación: Usa espacios para estructurar el código
• Multiplataforma: Funciona en Windows, Mac, Linux

Variables: Cajas para Guardar Cosas

📦 ¿Qué es una variable?

Una variable es como una etiqueta que apunta a valores en memoria. No necesitas declarar el tipo explícitamente - Python lo detecta automáticamente.

# Crear variables es súper fácil
nombre = "María"
edad = 25
ciudad = "Madrid"
# Usar las variables
print("Hola, soy", nombre)
print("Tengo", edad, "años")
